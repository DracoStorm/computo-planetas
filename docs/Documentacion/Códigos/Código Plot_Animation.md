# Código Plot_Animation

Aquí definimos una función que crea una animación para mostrar la trayectoria y las posiciones de dos planetas en un plano bidimensional.



## Librerías

*   [matplotlib.pyplot y matplotlib.animation](../librerias/Librería_MatPlotLib.md), Éstas permiten la creación de gráficos en dos dimensiones.



##  Funciones

1.   Función `update_plot()`:

Esta función se utiliza para actualizar los elementos en cada frame de la animación.
Toma como argumentos el número de frame actual, las trayectorias de los planetas, y los objetos de la trama que se deben actualizar.

2.  Función `plot_animation`:

La función crea una animación que muestra la trayectoria de dos planetas en un plano. Actualiza dinámicamente una visualización de la trayectoria orbital de los planetas.
